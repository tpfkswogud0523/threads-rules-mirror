# skill: 건강상식 정보 (주제 → 리서치 → 스레드) — v2

> **상속**: `../CLAUDE.md`(헌법)와 `../스타일가이드.md` v5(강제 교정규범)를 상속·구체화한다. 충돌 시 헌법·스타일가이드 우선(안전성→진정성→후킹력).
> **위상(2026-06-25 v2)**: 글 생성은 에이전트 팀(`../.claude/agents/`)이 분업으로 돌리고, 이 skill은 **주제 리서치 정보형 유형의 레시피**. v2는 스타일가이드 v5 권위론·R-rules·9피드백 반영.
> **참조**: 경험 양념은 `../경험뱅크.md`에서만. 거기 없는 경험은 지어내지 않는다.
> **차이점**: `블로그-재가공` skill은 *기존 블로그 글*을 가공한다. 이 skill은 **블로그 원문 없이 주제만 받아** 직접 리서치해서 정보형 스레드를 만든다.

## 0. 권위론 — 이 skill의 절대 원칙 (스타일가이드 v5 §0·§1-1)

> **다라는 본인 권위가 약하다.** 긴 설명·논증 글(자청식 롱폼·부업 명령형·의인화 설명)은 결을 안 쓴다. 이 skill로 만드는 일반 정보형은 **본인 권위 없이 작동하는 두 축**에서 벗어나지 않는다:
> ① **공감 상황 → 행동지침 넘버링** (R1·R2)
> ② **외부 권위 인용** (R6 캡쳐 — 논문·전문가 기사·공공기관 자료)

다음을 발견 즉시 폐기:
- "면역세포가 어떻게 일하는지 풀어 설명" 식 의인화·설명식 톤 (ref25 결) — 다라 권위 부족으로 못 받침
- "월 100만원·직빵·지금 당장 끊어·9번이 제일 중요" 자청식 강도 어법
- 본인 캐릭터·전문가 권위에 기댄 톤

## 1. 언제 쓰나 (트리거 / 입력)

- 입력 = **건강 주제 하나**(예: "야식 끊으면 달라지는 것", "수면과 면역"). 블로그 원문 없음.
- 출력 = 리서치 근거에 기반한 **정보형 스레드 1세트**(본문 + 댓글 N), 헌법 §15 형식.
- 본질은 **일방적 정보 전달**. 경험은 류마티스/면역 접점일 때만 한 줄 양념.

> **🕛 자동 수급 규칙 (예약 루틴 A — 매일 오전 7시)**: 매일 07:00 KST 자동 실행 → 주제 1개 → 정보형 1편 → Notion 큐에 상태=생성됨 적재(발행은 사람이 검수 후 **수동**).
> - **중복 회피(필수)**: 큐 기존 제목·주제 먼저 확인.

## 2. 파이프라인 (순서대로)

1. **주제 확정 + 핵심 메시지 1개.** "독자가 이 글에서 가져갈 한 줄".

2. **★R패턴 매칭 매트릭스로 글 형식 결정** (스타일가이드 v5 §16-7):

   | 주제 유형 | 추천 R 조합 |
   |---|---|
   | 즉시 실천 행동지침 (식사·수면·운동 루틴) | R1+R2+R3+R4 |
   | **공감 체크리스트 + 루틴 처방 ★다라 최적합** | R1+R2+R4 (+R6 권장) |
   | TOP N 압축 큐레이션 (식품·영양·약국템) | R1+R3+R4+R5 |
   | 두고두고 보는 데이터·비율표 | R5+R3+R4 |
   | 면역·세포 주제 (어렵게 풀지 말 것) | R1+R2+R3+R4 (+R6) — 신호·루틴 행동지침으로 |
   | 부정각도 (해로운 것 N가지) | R1+R2+R4 + 역순 옵션 |

3. **리서치 → 출처 확보 (필수)**.
   - 검색은 Firecrawl 우선 → 한도/오류 시 Playwright 폴백.
   - 신뢰 근거(연구·학술지·병원·공공기관) **2~3개** 확보. **출처 형식 = [핵심 압축 명칭]+연도+연구자**(스타일가이드 §2-1 ⑤·§16 R6).
     - ✅ "야식과 지방 대사 연구(Vujović et al., 2022)"
     - ❌ "(사람 실험, 2022)" — 어떤 연구인지 모름
   - `../경험뱅크.md` §5에 이미 있는 팩트면 재사용.

4. **R6 캡쳐 자료 확보** (해당 R 조합일 때):
   - 논문 요약/결과 캡쳐 또는 권위 의료기관 공식 자료 캡쳐.
   - 자료 도메인 일치(스타일가이드 §6) — 식사 후킹이면 식사 근거, 보충제 자료 X.
   - 캡쳐 시 출처·연도·연구자가 화면 안에 보이게.

5. **설계 4단(헌법 §8)**: 독자 / 핵심 욕구 / 예상 반박 3+해소 / 약속 베네핏 1개.

6. **본문(훅) 설계** (R1 공감 진입 + R2 결론 보류):
   - 독자가 "내 얘기네" 느낄 상황 1~3줄 + 후킹 (스타일가이드 §4).
   - 예: "생리보다 생리 전 일주일이 더 힘든 사람? / 예민한 게 아니야" (ref27)
   - **R2 본론 훅 + 댓글 결론 분리** ★사용자 명시 핵심: 본론에 결론·실천 안 풀기. 본론에서 다 풀면 댓글 안 봄.
   - **R3 본문 저장 CTA**: "저장해두고 봐" / "메모해놔" 잔잔하게. **자청식 강도("직빵·지금 당장·반드시") 절대 금지**.
   - **예고 금지(§3)**: "아래에서 풀게" X.
   - 통념이면 ban list(§5-1) 확인 → §5.5 5각도로 뒤집기. 못 뒤집으면 폐기.

7. **댓글(전개)** — R 조합대로:
   - 카드별 소주제 1개. **R1 넘버 행동지침** 또는 **R5 분류표·비율표**.
   - **R6 캡쳐 카드**: "👇 [핵심 압축 명칭]+연도+연구자" + 이미지.
   - **인용 톤(§13-1)**: 다라 직접 체험 아닌 효과는 "~한다고 하더라" / "~다는 연구가 있더라". 1인칭 단정 금지.
   - **노션 9피드백(§2-1) 자기 체크 통과 필수**.

8. **착지(마지막 카드)** — **R4 전달 가치 한 줄 필수**:
   - "○○ 고생하는 친구한테도 보내줘" / "같이 챙기는 사람 있으면 공유해줘".
   - 안전선은 §10-2 결과 약속 강도 매트릭스로:
     - 의료 위험 정보 → 안전선 변주 풀에서 (§10-1)
     - 생활 습관 정보 → **안전선 생략** + 결과 약속 강하게("한 끼만 바꿔도 충분" 식)

9. **포맷 판단**(헌법 §14): "알아두면 좋은 N가지"·통계·리스티클이면 **이미지 카드**(헌법 §11 규격) 적극 고려. R5·R6 적용 글은 이미지 카드가 강함.

### ⭐ 장수 규칙 (v2 강화)
- 정보형 = **5장 이하** (스타일가이드 §1-2). 본문 1 + 댓글 3~4 기본.
- 본인 권위 부족 → 길게 설명하지 말 것. 짧고 직관적으로.
- 카드 글자수 = §6-2 (본문≤300자, 댓글≤350자, 마지막≤400자). **실질 목표 공백 제외 100자 내외**(사용자 2026-06-25 의견).

### ⭐ 경험 후기 → 링크 퍼널 (선택 변형)
이 skill은 정보형이라 본문에서 정보를 충분히 주는 게 기본이지만, **깊은 how-to·전체 루틴은 블로그로 보내는 퍼널** 마무리를 선택적으로 쓴다.
- 적용 조건: 류마/면역 접점이라 `경험뱅크.md` atom으로 경험을 진짜로 댈 수 있을 때.
- "방법은 길어서 프로필 블로그에 정리해뒀어" + 참여 질문. **알맹이 없는 낚시 금지**.

## 3. 안전·신뢰 규칙 (정보형에서 특히 엄격)

- **출처 없는 단정 통계 금지**(헌법 §13). 숫자는 §7 가드 — 검증된 것만, 아니면 뭉뚱그림.
- 단정어("무조건 낫는다") 금지 → "~라고 알려져 있어 / 한 연구에선 ~".
- 병명·진단 단정 금지, 약 폄하·중단 권유 금지(헌법 §9).
- 면책문구 따로 X, **톤에 녹임**: "사람마다 다르더라", "심해지면 전문의 꼭".
- **⭐v2 자청식 강도 어법 금지** (§16 R3): "직빵·필수·반드시·지금 당장·9번이 제일 중요" — 다라 톤 깸. §15-E bash 게이트가 자동 차단.
- **⭐v2 의인화 긴 설명 금지** (§1-1): "대식이가 뭘 한다" 식 면역세포 풀이 X. 면역 주제도 신호·루틴 행동지침으로(§16-7 매트릭스).
- **경험 양념**: 류마/면역 접점이면 `경험뱅크.md` atom 한 줄. 접점 없으면 **"내가 찾아보니"** 톤.

## 4. 출력 형식 (헌법 §15 + v2 확장)

```
[설계]   독자 / 욕구 / 예상반박3+해소 / 약속 베네핏
[주제]   + R패턴 매칭 매트릭스로 결정한 R 조합 (예: R1+R2+R3+R4+R6)
[글 형식]  공감 체크리스트형 / TOP N 큐레이션형 / 비율표형 / 부정각도 역순형 등
[출처]   확보한 근거 2~3개 — [핵심 압축 명칭]+연도+연구자 형식
[R6 캡쳐] 첨부 계획 (있으면)
[포맷]   텍스트 / 이미지카드 (+사유)

[본문 1/N]  (R1 공감 진입 + R2 결론 보류 + R3 저장 CTA 잔잔하게)
[댓글1 2/N] (R6 캡쳐 / 출처 형식 / 인용 톤)
[댓글2 3/N] (R1 행동지침 1)~N) / 인용 톤)
[마지막 N/N] (R4 전달 가치 + 안전선 매트릭스 적용)

[정량 게이트 결과] §15 run_v4_gate PASS/FAIL
[9피드백 자기 체크] §2-1 ①~⑨ 전부 PASS
[R-rules 적용 확인] R1=Y, R2=Y, R3=Y, R4=Y, R5=N, R6=Y 식
```
> N은 §2 장수 규칙대로 **최대 5장**.

## 5. 미니 예시 — 주제 "잠과 면역" (텍스트형, 축약, R1+R2+R3+R4+R6 적용)

```
[본문] 1/4 (R1 공감 + R2 결론 보류 + R3 저장 CTA)
요즘따라 자꾸 잔병치레 하는 사람?
"좀 덜 자도 괜찮아" 하고 넘기지만
잠은 그냥 쉬는 시간이 아니야
오늘 밤부터 30분만 바꿔봐
저장해두고 봐

[댓글1] 2/4 (R6 캡쳐 + 출처 형식 + 인용 톤)
적게 자면 감기 확률이 4.2배
6시간 이하로 자면
7시간 이상보다 감기 확률 4.2배 높았다고 하더라
👇 수면 시간과 감기 감수성 코호트 연구
(Prather et al., Sleep, 2015)
[캡쳐 이미지]

[댓글2] 3/4 (R1 행동지침)
오늘부터 해볼 3가지
1) 평소보다 30분 일찍 누워보기
2) 자기 전 미지근한 물 한 잔
3) 침실 온도 18~20도

[마지막] 4/4 (R4 전달 가치 + 안전선 생략 — 생활습관 정보)
잠 잘 못 자는 친구한테도 보내줘
오늘 밤부터 30분만 일찍 누워봐, 진짜 달라진다더라
```

- [ ] 출처 없는 단정 통계 / 지어낸 숫자 없나 (헌법 §7·§13)

- [ ] (퍼널형이면) 경험 후기가 경험뱅크 근거 있는 진짜인가 / 본문 자체로 가치 있나(낚시 X)

### 6-B. R-rules (§16)
- [ ] R패턴 매칭 매트릭스(§16-7)로 R 조합 결정했나
- [ ] **R1 공감 진입 → 넘버 행동지침** 적용?
- [ ] **R2 본론 훅 + 댓글 결론 분리** (본문에 결론·실천 안 풀기)?
- [ ] **R3 본문 저장 CTA** ("저장해두고 봐" 잔잔하게, **자청식 강도 X**)?
- [ ] **R4 전달 가치 한 줄** (마지막 카드)?
- [ ] **R5 저장형 압축 포맷** (적용 시): 분류표·비율표·TOP N로 시각 분리?
- [ ] **R6 전문가 인용 캡쳐** (적용 시): 출처 [핵심 압축 명칭]+연도+연구자?

### 6-C. 노션 9피드백 (§2-1)
- [ ] ① "A가 아니야 → B야" 논리 일관성?
- [ ] ② 주체(무엇이/무엇을) 명시?
- [ ] ③ "~더라" 연속 3회 이상 없음?
- [ ] ④ 합의 전제 반복 없음?
- [ ] ⑤ 출처 [핵심 압축 명칭]+연도+연구자?
- [ ] ⑥ 예고 문구 자연스럽고 다음 카드와 일치?
- [ ] ⑦ 훅 베네핏 → 다음 카드에서 바로 답?
- [ ] ⑧ "~한다는 말" 정리체 없음?
- [ ] ⑨ 불필요한 따옴표 강조 없음?
